# 🦄 Unicorn Startups Data Analysis

## 📌 About the Project
This project performs an exploratory data analysis (EDA) on a dataset of unicorn startups (companies valued at over $1 billion).

The goal is to identify patterns and insights related to countries, industries, and company valuations.

---

## 📊 Objectives
- Analyze the distribution of unicorn companies by country  
- Identify the most common industries  
- Evaluate valuation trends  
- Explore the growth of unicorns over time  

---

## 🛠 Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## 📈 Analysis Performed
- Data cleaning and preprocessing  
- Handling missing values  
- Column renaming  
- Conversion of data types  
- Feature engineering (year and month extraction)  
- Aggregation and grouping analysis  

---

## 📷 Visualizations
The project includes:
- Bar chart of sectors  
- Pie chart of top countries  
- Line chart of unicorn valuations

---

## 📂 Dataset
The dataset contains information about unicorn startups up to 2021, including:
- Company name  
- Country and city  
- Industry  
- Valuation  
- Date joined  

---

## 🚀 Conclusion
- The United States leads in the number of unicorn companies  
- Technology-related industries dominate the market  
- There has been significant growth in unicorn creation over recent years
